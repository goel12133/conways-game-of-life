# Conway's Game of Life

## Problem Description

The simulation models a cell-based automaton in which cells evolve according to four rules.
Users can place cells manually, pause the simulation, step through generations, and load several well-known starting arrangements.

## Setup

1. Install Python 3.
2. Install pygame 

```bash
pip install -r requirements.txt
```

## Run Instructions

```bash
python main.py
```

## Controls

- Space: Pause / Resume
- N: Advance one generation while paused
- Mouse Click: Add or remove a cell
- 1: Glider
- 2: Blinker
- 3: Toad
- 4: Pulsar
- 5: Gosper Glider Gun
